# Monit-Online

这是一个基于 [Monit](https://github.com/fzf404/Monit) 二次开发的 **可联网版本 Monit**，支持在线同步。

## 项目介绍

Monit-Online 在原版 Monit 的基础上进行扩展，目前仅保留了 **Todo** 功能，并新增了 **联网同步** 功能，方便用户在多设备之间同步和管理待办事项。

## 功能特性

- ✅ 保留 Todo 功能
- ☁️ 支持联网同步，跨设备管理
- 💻 基于原版 Monit 改造，保持轻量简洁
